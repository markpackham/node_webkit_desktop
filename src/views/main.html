NW.js (Node-Webkit) Desktop App

learned from an exercise on NW.js https://www.youtube.com/watch?v=5UsGnjPYxLU
npm run dev in root folder to see NW at work
npm run prod to create "Breakout_Game" exe files in the dist folder

How To Play:

Use your right and left keys to move the paddle to bounce the ball up and break the blocks.

If you miss the ball, your score and the blocks will reset.
